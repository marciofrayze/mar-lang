# Contributing

Thank you for your interest in Mar.

At the moment, this project is not accepting pull requests.

Mar is still in a highly experimental stage, and the language, runtime, tooling, and overall structure are changing frequently. Keeping development centralized helps the project move faster while these foundations are still shifting.

The best ways to help right now are:

- test the project in real scenarios
- use it and share feedback
- help spread the word
- report bugs and rough edges you find

Bug reports and practical feedback are especially valuable while Mar is still in an early stage.

If you find a bug or want to share feedback, you can:

- open an issue
- contact me directly at marcio@segunda.tech
